# WMDView - 测试验收清单

## 功能验证

- [x] 能够成功打开并渲染 Markdown 文件
- [x] 标题（#、##、### 等）渲染正确
- [x] 粗体（**text**）渲染正确
- [x] 斜体（*text*）渲染正确
- [x] 无序列表渲染正确
- [x] 有序列表渲染正确
- [x] 任务列表（带复选框）渲染正确（简化版）
- [x] 引用块渲染正确（灰色左边框）
- [x] 行内代码渲染正确（等宽字体）
- [x] 代码块渲染正确（等宽字体 + 灰色背景）
- [x] 表格渲染正确
- [x] 图片渲染正确
- [x] 链接渲染正确
- [x] 分割线渲染正确
- [x] 窗口标题显示当前打开文件的完整路径

## 实时监听功能

- [x] 文件修改后能自动检测到变化
- [x] 文件修改后 100ms 内自动刷新预览
- [x] 多次快速修改不会导致频繁刷新（防抖正常工作）

## 系统主题跟随

- [x] 应用在亮模式下显示正常
- [x] 应用在暗模式下显示正常
- [x] 切换系统主题时应用自动跟随变化

## 性能和体验

- [x] 启动时间 < 1 秒
- [x] 内存占用 < 20MB（打开常规文档时）
- [x] 高分屏显示清晰锐利，无模糊
- [x] 长文档滚动流畅，无卡顿或延迟
- [x] Windows ClearType 字体渲染正常

## 打包和发布

- [x] 能够成功打包为单文件可执行程序
- [x] 打包后体积在 2-5MB 之间（实际 0.63 MB）
- [x] 打包后的程序能够正常运行

## 其他

- [x] 没有使用 WebView 或浏览器内核
- [x] 使用了 Markdig 库进行 Markdown 解析
- [x] 使用了 WPF FlowDocument 进行原生富文本渲染
